Demo Example

Setup

PC Node
  1. Install Java (>8 or >11 if one wants to use the MapEditor);
  2. Install docker.
  3. Install gradle (>= 7.4).
  4. (Optional, if one doesn't have an Android device)
    • Install an Android Emulator (guide);
    • Install Flutter (guide)
  5. From the latest release on our GitHub repository, download:
    • VirtualRobot docker image (virtualRobotOnly4.0.yaml).
    • WasteService Core & WasteService RPi ZIP archives.
    • SmartDevice APK (smart_device.apk).
    NB: file names can vary a bit (sprint number, version number). Use the latest version available.
NB: to download a ZIP containing the basics needed to run the system, download Sprint2_WasteServiceBCR.zip
Raspberry Pi Node
  1. SSH must be installed and usable (see raspi-config);
  2. VNC could be another useful tool;
  3. make sure Python is installed;
  4. make sure Java (>8) is installed;

Execution

  1. Run the virtual robot docker image;
  2. Open a browser at http://localhost:8090/;
  3. Run the WasteService Core contexts (WasteService, TransportTrolley and Robot);
  4. Run the WasteService RPi context
