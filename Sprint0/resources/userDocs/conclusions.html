Sprint Conclusions

Summary

This first Sprint (Sprint0) outlined the following macro elements of the project:
  1. Core-business: involves the chain of the main nodes of the system (smart device, waste service, transport trolley, basic robot)
  2. Monitoring: involves the GUI
  3. Alarm: involves the raspberry device

Work Plan

Sprint1 - Core Business

The Sprint1 will focus on the Core business, which is also the first requirement, and is going to include:
  • Dummy simulator of TruckDriver + SmartDevice.
  • WasteService: problem analysis regarding a model for "advanced" interactions, waste service area in-depth analysis, project and implementation of a prototype.
  • TransportTrolley: problem analysis, project and prototyping of the element that will control the robot movements.
  • BasicRobot: the software is ready and serviceable, so it just has to be integrated in the project.
