SmartDevice Simulator

SmartDevice Simulator is a mobile/desktop compatible application, which will be used just for testing purposes, to simulate the interaction between the TruckDriver and the WasteService: It will allow to send store requests, and display the response.

Echo Waste Server

Echo Waste Server is a mobile/desktop application that simulates a dummy WasteService and acts like an echo server: it accepts TCP connections and reply to store_requests with load_accepted / load_rejected, updating the capacity of containers and logging messages.

Its purpose is to test the SmartDevice directly and completely on smartphone, without having to connect it to another device.
