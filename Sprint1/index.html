Sprint 1

Introduction

Our team must project and develop a software system to manage a waste-disposal service, as described in Final Theme 2022.

Scrum Goal

We must provide the customer an executable demo, involving the core-business of the system (requirements 1 & 2). The core-business is the chain of the main nodes of the system (smart device, waste service, transport trolley, basic robot).

Work Plan

  • Analyze what's needed to implement the WasteService Core-Business
  • Find and address any problem related to the requirement (find difficulties, make considerations, suggest solution/s)
  • Implement at least a simulator for the SmartDevice, capable of interacting with the WasteService
  • Implement the WasteService software, including all of its components
  • Test the system
  • Create a demo prototype of the of the system
NB: make use of the software provided by our software house.

Motto

There is no code without a project, there is no project without problem analysis, there is no problem without requirements.
