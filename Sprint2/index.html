Sprint3

Sprint 2

Introduction

Our team must project and develop a software system to manage a waste-disposal service, as described in Final Theme 2022.

Scrum Goal

We must develop a warning/alarm device using a RaspberryPi, a Led and a Sonar, and implement a stop/resume feature for the DDR Robot (requirement 4). We also have to provide the customer an executable demo of the component being integrated with the rest of the system.

Work Plan

  • Interactions:
    • SmartDevice and WasteService
    • WasteService and TransportTrolley
  • WasteService: handling of stop/resume messages
  • Models & TestPlans for each interaction
  • Demonstrative Prototype of the whole new system
NB: eventually make use of the software provided by our software house.

Motto

There is no code without a project, there is no project without problem analysis, there is no problem without requirements.
